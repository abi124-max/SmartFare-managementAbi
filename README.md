# Smart Fare Management System - Customer App

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Java (Spring Boot)
- Database: MySQL
- Runtime: Kiro AI

## Phase 1 Features
1. Location selection (From/To)
2. Bus availability display
3. Bus selection and UPI payment
4. Digital ticket with QR code
5. Mobile ticket storage
